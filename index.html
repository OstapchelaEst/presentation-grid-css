How to create lines and columns?

grid-template-columns - defines line names and grid column sizes

/* grid-template-columns: 100px 100px 100px; */

Will create three columns for 100px

grid-template-rows - defines the line names and stripe sizes of the grid rows

/* grid-template-rows: 20% 20% auto; */

Will create two rows by 20% from parent size and one row for the rest of the width

                  .grid-container {
                     display: grid;
                     grid-template-columns:auto 30% 3rem ;
                     grid-template-rows: 25vh auto 100px;
                     gap: 10px;
                  }

justify-content

This property allows us to align elements with respect to the main axis.


Has such values as

  • start
  • center
  • end
  • space-between
  • space-around
  • space-evenly

justify-content: start

                  .grid-container {
                     display: grid;
                     background-color: rgba(129, 129, 129, 0.321);
                     grid-template-columns:repeat(3, 200px);
                     grid-template-rows:repeat(3, 55px);

                     gap: 10px;

                     justify-content: start;
